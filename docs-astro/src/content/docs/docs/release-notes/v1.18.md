## v1.18.0

### New Containers

- [Scrutiny](https://github.com/AnalogJ/scrutiny) is now available to monitor drive SMART status
- [Beszel](https://github.com/henrygd/beszel) is now available for host monitoring

### Removed container

Removed Huntarr since it has been removed from everywhere

### Other changes

If using SSL/TLS, the `websecure` entrypoint is now the default, this will halve the amount of routers for Traefik
